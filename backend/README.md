# Verdify Backend

Green transportation routing backend for Malaysia.

## Run
1. Set `PORT` if needed.
2. `go run .`
3. `GET /health`

## Current Scope
- Go backend skeleton
- Health endpoint
- Docs for AI-guided build flow

## Required Stack
- Go
- Genkit
- Google Cloud
- Vertex AI
- Firebase Firestore
- Google Maps API
- Gemini via Vertex AI
